TASK IN ONE SENTENCE
Find and fill the largest square on a 2D map that avoids obstacles.

CRITICAL REQUIREMENTS
1. ✓ Parse 4-value header: <rows> <empty_char> <obstacle_char> <full_char>
2. ✓ Validate strictly: characters distinct, rows equal, row count matches
3. ✓ Find largest square (max area, width = height)
4. ✓ Tie-breaking: pick top-left corner position
5. ✓ Output: modified map to stdout, "map error\n" to stderr on failure
6. ✓ Support: file input, stdin input, multiple maps
7. ✓ Memory: allocate dynamically, free everything

VALIDATION CHECKLIST (IF ANY FAILS → "map error")
☐ Header has exactly 4 space-separated values
☐ First value is integer > 0
☐ Three characters are printable (ASCII 32-126)
☐ Three characters are all different (no duplicates)
☐ All map rows have identical length
☐ Row count matches header value
☐ Each row ends with newline
☐ Only empty/obstacle chars in map (no other chars)

COMMON MISTAKES (AVOID THESE!)
✗ Not checking that 3 chars are distinct (e≠o, e≠f, o≠f)
✗ Not validating all rows same length
✗ Wrong tie-breaking (should prefer top, then left)
✗ Memory leaks (not freeing getline buffer, map)
✗ Using printf instead of fprintf(stderr) for errors
✗ Missing newline after output
✗ Not supporting stdin
✗ Incorrect square position calculation
