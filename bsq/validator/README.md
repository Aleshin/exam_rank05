# Валидатор BSQ (компактный)

Это отдельная утилита, которая **только проверяет** входные данные по правилам проекта.
Ничего не вычисляет и не рисует квадрат — только валидирует.

## Что проверяется

1. Заголовок в формате: `rows empty obstacle full` + перевод строки
2. `rows > 0`
3. Символы `empty`, `obstacle`, `full` — **разные** и печатные (ASCII 32..126)
4. Ровно `rows` строк карты
5. Все строки одинаковой длины
6. Только символы `empty` и `obstacle`
7. После карты **нет лишних строк**

## Вывод

- Если всё корректно → печатает `ok` в stdout
- Если ошибка → печатает `map error` в stderr

## Компиляция

```
make
```

## Использование

```
./validator map.txt
cat map.txt | ./validator
./validator map1.txt map2.txt
```

## Короткое объяснение коду

- `read_header()` читает и валидирует заголовок.
- `validate_map()` читает карту построчно, проверяет длину и символы.
- После чтения нужного количества строк проверяет, что больше данных нет.

Код максимально компактный, но с поясняющими комментариями на русском.
