КОМПАКТНОЕ РЕШЕНИЕ BSQ - КРАТКОЕ ОБЪЯСНЕНИЕ
============================================

СТРУКТУРА (t_data)
───────────────────
- rows, cols: размеры карты
- map: двумерная карта
- e, o, f: символы (empty, obstacle, full)
- h: высоты для DP (динамическое программирование)
- sz, sr, sc: размер и позиция квадрата


АЛГОРИТМ В 3 ШАГА
──────────────────

1. read_header() + read_map()
   Парсим заголовок, валидируем, читаем карту

2. find_square()
   Для каждой строки:
   - Увеличиваем высоту h[j] если пусто, иначе обнуляем
   - Для каждого столбца проверяем все возможные квадраты
   - Сравниваем с максимальным (sz, sr, sc)

3. print_map() + free_data()
   Выводим карту с заполненным квадратом и освобождаем память


ОСНОВНЫЕ ПРОВЕРКИ
──────────────────
✓ Парсинг: 4 значения, integer > 0, символы 32-126, разные
✓ Карта: все строки одной длины, только e/o символы
✓ Память: все malloc/free на месте
✓ Вывод: stdout для результата, stderr для ошибок


КОМПАКТНОСТЬ
─────────────
- Один файл (173 строки)
- Нет лишних функций
- Минимум переменных
- Прямой алгоритм DP


ЗАПОМНИТЬ
──────────
1. Читай заголовок с fscanf
2. Проверяй что 3 символа разные (e != o, e != f, o != f)
3. Для каждой строки обновляй высоты
4. Проверяй квадраты в гистограмме
5. Выводи результат или ошибку
6. Освобождай память
